📥 データセットの準備

GitHub のファイルサイズ制限により、このリポジトリにはデータセットは含まれていません。

ステップ1：元データをダウンロードする

以下の Kaggle ページからデータセットをダウンロードしてください。

Kaggle リンク：
https://www.kaggle.com/datasets/brianmcgloughlin/japanese-housing-prices-2005-2024

---

ステップ2：フォルダを作成する

プロジェクトフォルダ内に、以下のフォルダを作成してください（存在しない場合）。

data/
└── raw/

---

ステップ3：データセットを配置する

ダウンロードした CSV ファイルを "data/raw/" フォルダへコピーしてください。

例：

data/
└── raw/
    └── all_prefecrure......csv

«注意： ファイル名を変更する場合は、"src/eda.py" 内のファイルパスも合わせて変更してください。»

---

🧹 クリーンデータ（cleaned.csv）の作成

前処理スクリプトを実行すると、クリーンなデータセットが自動的に作成されます。

ステップ1

プロジェクトフォルダでターミナルを開きます。

ステップ2

以下のコマンドを実行してください。

python src/eda.py

---

ステップ3

スクリプトは以下の処理を実行します。

- 元データの読み込み
- データのクリーニング
- 欠損値やデータ形式の処理
- 前処理済みデータを保存

生成されるファイル：

data/
└── cleaned.csv

"cleaned.csv" が作成された後、すべての分析スクリプトを実行できます。

---

▶️ プロジェクトの実行

"cleaned.csv" の作成後、分析スクリプトを実行してください。

例：

python src/your_analysis_script.py

"your_analysis_script.py" を実際に実行したい分析スクリプト名に置き換えてください。
